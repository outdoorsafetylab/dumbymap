# Minimal Setup DumbyMap can be loaded with a single `<script>` tag pointing to the bundled `dist/dumbymap.mjs`. No HTML boilerplate required — write plain Markdown directly in the file and the auto-init detects the empty script body, injects the stylesheets, and calls `generateMaps`. ## How it works When `dumbymap.mjs` is loaded via `<script type="module" src="...">` with no inline body, it: 1. Collects all text nodes from `<body>` as the Markdown source. 2. Injects `css/dumbymap.css` and `css/style.css` from the same directory. 3. Calls `generateMaps(document.body)` automatically. ## Map example ```map use: Leaflet center: [121.5645, 25.0338] zoom: 12 ```
